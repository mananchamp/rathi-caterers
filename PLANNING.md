# Rathi Caterers Website - Project Planning

## 1. Project Overview
- **Project Name:** Rathi Caterers Website
- **Type:** Corporate/Business Website (Single Page with smooth scrolling)
- **Purpose:** Showcase catering services, build trust, generate leads
- **Target Audience:** Corporate clients, families planning events, wedding planners
- **Reference Site:** https://goodfoodconcept.in/

---

## 2. Color Palette (Modern Aesthetic)

### Primary Colors
| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Deep Charcoal | `#1A1A2E` | Primary background, headers |
| Warm Gold | `#D4AF37` | Accent, CTAs, highlights |
| Soft Ivory | `#FAF9F6` | Light background, text areas |

### Secondary Colors
| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Slate Gray | `#4A4A5A` | Subheadings, secondary text |
| Terracotta | `#C9705A` | Secondary accent, hover states |
| Sage Green | `#7D8471` | Decorative elements, icons |

### Neutrals
| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Pure White | `#FFFFFF` | Cards, text on dark |
| Light Gray | `#F5F5F5` | Section backgrounds |
| Dark Text | `#2D2D2D` | Body text |

---

## 3. Typography

- **Headings:** Playfair Display (Serif) - Elegant, premium feel
- **Body:** DM Sans (Sans-serif) - Clean, modern readability
- **Accent:** Cormorant Garamond (Italic) - For quotes/taglines

---

## 4. Website Structure

### 4.1 Header/Navigation
- Logo (left)
- Nav links: Home, Services, About, Gallery, Contact
- CTA Button: "Enquire Now"
- Mobile: Hamburger menu

### 4.2 Hero Section
- Full-screen background with food imagery (overlay)
- Headline + Subheadline
- Primary CTA: "View Our Services"
- Secondary CTA: "Contact Us"

### 4.3 About Section
- Company intro (brief history, experience)
- Years in business badge
- Mission statement
- Image + text layout

### 4.4 Services Section
- 4 main service cards:
  1. Wedding Catering
  2. Corporate Catering
  3. Party Events
  4. Outdoor/Custom Events
- Each card: Icon, title, brief description, "Learn More" link

### 4.5 Why Choose Us Section
- Key differentiators (3-4 points)
- Stats: Events Catered, Years Experience, Happy Clients

### 4.6 Gallery Section
- Image grid (6-9 images)
- Hover effect with category label

### 4.7 Testimonials Section
- Client testimonials carousel
- Star ratings

### 4.8 Contact Section
- Contact form (Name, Email, Phone, Event Type, Message)
- Contact info (Phone, Email, Address)
- Google Map embed placeholder

### 4.9 Footer
- Logo
- Quick links
- Social media icons
- Copyright

---

## 5. Technical Specifications

- **Framework:** Vanilla HTML/CSS/JS (or React/Next.js)
- **Responsive:** Mobile-first, breakpoints at 768px, 1024px
- **Animations:** Subtle fade-ins, hover effects, smooth scroll
- **Images:** Lazy loading, optimized formats
- **Performance:** Lighthouse score target 90+
